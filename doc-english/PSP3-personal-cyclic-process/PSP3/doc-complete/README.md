# Complete Documentation — PSP3 (Cyclic Personal Process)

The complete PSP3 set — incremental cyclic development with consolidated measurement and quality.

## Structure
- project/ — Requirements, description, overview and roadmap
- planning/ — Master plan, overall schedule, global estimate and iteration strategy
- iterations/ — One folder per cycle (iteration-01..04), each with planning, estimates, logs, reviews and an iteration report
- quality/ — Plan, checklists, defect classification and history, metrics, defect analysis, causal analysis and improvements
- measurement/ — Time, defect, productivity and estimate summaries
- reports/ — Project status, milestone report, final report and lessons learned
- personal-process/ — Coding standard, process improvements and development flow
