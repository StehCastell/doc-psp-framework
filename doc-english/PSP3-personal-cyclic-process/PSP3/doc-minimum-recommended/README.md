# Recommended Minimum Documentation — PSP3 (Cyclic Personal Process)

PSP3 scales the PSP to larger programs by splitting development into cycles (iterations), each with its own planning, logs and reviews.

## Structure
- project/ — Requirements and project description
- iterations/ — One folder per cycle (iteration-01, iteration-02), each with work plan, effort estimate, time log and defect log
- quality/ — Design and code checklists and defect classification
