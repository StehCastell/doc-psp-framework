# Complete Documentation — PSP1 (Personal Planning)

The complete PSP1 set.

## Structure
- project/ — Requirements, project description and notes
- planning/ — Size estimate, effort estimate, work plan, schedule and tracking
- measurement/ — Time log, defect log, software size and measurement summary
- personal-process/ — Coding standard, development flow, process improvements and lessons learned
- configuration/ — Change control
- reports/ — Planned vs actual report and closure report
