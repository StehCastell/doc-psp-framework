# Recommended Minimum Documentation — PSP2.1 (Personal Quality)

PSP2.1 deepens quality with a standardized defect classification (Defect Type Standard) and defect analysis.

## Structure
- project/ — Requirements and project description
- planning/ — Size estimate, effort estimate and work plan
- measurement/ — Time log, defect log and software size
- quality/ — Design and code checklists, design review, code review, defect classification and defect analysis
