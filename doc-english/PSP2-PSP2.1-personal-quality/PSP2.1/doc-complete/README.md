# Complete Documentation — PSP2.1 (Personal Quality)

The complete PSP2.1 set.

## Structure
- project/ — Requirements, project description and notes
- planning/ — Size estimate, effort estimate, work plan and schedule
- measurement/ — Time log, defect log, software size and measurement summary
- quality/ — Checklists, design and code reviews, quality plan and report, defect classification and history, quality metrics, defect analysis, causal analysis and quality improvements
- personal-process/ — Coding standard, process improvements and lessons learned
- analysis/ — Productivity history, estimate history and planned vs actual
- reports/ — Closure report (postmortem)
