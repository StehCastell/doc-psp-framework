# 🔁 Process Area — Iterations

> This folder is **self-contained**: everything you need to understand the area is here.

## 📌 What this area is in PSP
PSP3 cyclic development: repeat the full process in iterations to scale to larger projects.

## 🎯 What problem it solves
Solves **applying PSP to large projects** that do not fit a single cycle.

## ✅ What belongs in this area
- Per-iteration work plan
- Per-iteration estimates
- Per-iteration reviews
- Iteration report

## 🚫 What does NOT belong in this area
- Process definition (→ personal-process)
- Final consolidation (→ reports)
