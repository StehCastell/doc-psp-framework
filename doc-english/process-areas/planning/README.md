# 📅 Process Area — Planning

> This folder is **self-contained**: everything you need to understand the area is here.

## 📌 What this area is in PSP
Personal size and time estimates (PROBE method) and schedule.

## 🎯 What problem it solves
Solves **guessed estimates** that blow deadlines.

## ✅ What belongs in this area
- Size estimate (PROBE)
- Effort/time estimate
- Schedule
- Work plan

## 🚫 What does NOT belong in this area
- Raw data (→ measurement)
- Estimate analysis (→ analysis)
