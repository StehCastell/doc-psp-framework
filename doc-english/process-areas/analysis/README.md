# 🔬 Process Area — Analysis

> This folder is **self-contained**: everything you need to understand the area is here.

## 📌 What this area is in PSP
Turns measurement into insight: estimate accuracy, productivity, trends.

## 🎯 What problem it solves
Solves **having data but not learning from it**.

## ✅ What belongs in this area
- Estimate analysis
- Productivity analysis
- Defect analysis
- Trend analysis

## 🚫 What does NOT belong in this area
- Collection (→ measurement)
- Improvement actions (→ personal-process)
