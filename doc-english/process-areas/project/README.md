# 📁 Process Area — Project

> This folder is **self-contained**: everything you need to understand the area is here.

## 📌 What this area is in PSP
Holds the data and plan of each personal project (summary, description).

## 🎯 What problem it solves
Solves **scattered, unconsolidated project data**.

## ✅ What belongs in this area
- Project description
- Project summary
- Consolidated project data

## 🚫 What does NOT belong in this area
- Detailed estimates (→ planning)
- Final reports (→ reports)

## 📈 How it evolves from PSP0 to PSP3

| Stage PSP | Evolution of the area |
|---|---|
| Stage PSP0 — Baseline | Project summary and base data. |
| Stage PSP0.1 — Standard and Size | Project with coding standard. |
| Stage PSP1 — Personal Planning | Project with estimates. |
| Stage PSP1.1 — Schedule and Earned Value | Project with schedule. |
| Stage PSP2 — Personal Quality | Project with quality plan. |
| Stage PSP2.1 — Design and Verification | Project with verified design. |
| Stage PSP3 — Cyclic Process | Project developed in cycles. |
