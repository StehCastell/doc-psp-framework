# ✅ Process Area — Quality

> This folder is **self-contained**: everything you need to understand the area is here.

## 📌 What this area is in PSP
Personal defect management: code and design reviews, checklists, quality plan.

## 🎯 What problem it solves
Solves **defects escaping to test and the customer**.

## ✅ What belongs in this area
- Code review
- Design review
- Review checklists
- Quality plan
- Defect management

## 🚫 What does NOT belong in this area
- Raw measurement (→ measurement)
- Causal analysis (→ analysis)
