# 📄 Process Area — Reports

> This folder is **self-contained**: everything you need to understand the area is here.

## 📌 What this area is in PSP
Communicates results: test report, cycle summary, final report.

## 🎯 What problem it solves
Solves **not closing the learning loop** at the end of the work.

## ✅ What belongs in this area
- Test report
- Measurement summary
- Final project report

## 🚫 What does NOT belong in this area
- Data collection (→ measurement)
- Analysis (→ analysis)

## 📈 How it evolves from PSP0 to PSP3

| Stage PSP | Evolution of the area |
|---|---|
| Stage PSP0 — Baseline | Test report and cycle summary. |
| Stage PSP0.1 — Standard and Size | Reports with size data. |
| Stage PSP1 — Personal Planning | Estimation report. |
| Stage PSP1.1 — Schedule and Earned Value | Planning and earned-value report. |
| Stage PSP2 — Personal Quality | Reviews and defects report. |
| Stage PSP2.1 — Design and Verification | Design quality report. |
| Stage PSP3 — Cyclic Process | Consolidated iterations report. |
