# 🧭 Process Area — Personal Process

> This folder is **self-contained**: everything you need to understand the area is here.

## 📌 What this area is in PSP
Defines and improves **your** way of working: scripts, phases and improvement proposals (PIP). It is the heart of PSP.

## 🎯 What problem it solves
Solves working **without a defined personal method**, which prevents measuring and improving.

## ✅ What belongs in this area
- Personal process definition (scripts/phases)
- Coding standard
- Process improvement proposals (PIP)
- Process notes

## 🚫 What does NOT belong in this area
- Estimates (→ planning)
- Measurement data (→ measurement)
