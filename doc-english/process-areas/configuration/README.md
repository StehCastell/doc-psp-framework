# 🗂️ Process Area — Configuration

> This folder is **self-contained**: everything you need to understand the area is here.

## 📌 What this area is in PSP
Version control of your personal artifacts (code, data, documents).

## 🎯 What problem it solves
Solves **losing history and versions** of personal work.

## ✅ What belongs in this area
- Personal version control
- Artifact organization

## 🚫 What does NOT belong in this area

- Reports (→ reports)
