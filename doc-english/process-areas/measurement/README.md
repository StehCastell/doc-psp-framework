# 📊 Process Area — Measurement

> This folder is **self-contained**: everything you need to understand the area is here.

## 📌 What this area is in PSP
Personal data collection: time, size and defects. The objective basis of PSP.

## 🎯 What problem it solves
Solves **not knowing where time goes or where you err**.

## ✅ What belongs in this area
- Time log
- Defect log
- Size measurement
- Quality metrics (yield, A/FR)

## 🚫 What does NOT belong in this area
- Estimates (→ planning)
- Data analysis (→ analysis)

## 📈 How it evolves from PSP0 to PSP3

| Stage PSP | Evolution of the area |
|---|---|
| Stage PSP0 — Baseline | Time and defect measurement. |
| Stage PSP0.1 — Standard and Size | Adds size measurement. |
| Stage PSP1 — Personal Planning | Size data for estimation. |
| Stage PSP1.1 — Schedule and Earned Value | Earned value measurement. |
| Stage PSP2 — Personal Quality | Defect measurement by phase. |
| Stage PSP2.1 — Design and Verification | Quality metrics (yield, A/FR). |
| Stage PSP3 — Cyclic Process | Consolidated measurement across iterations. |
