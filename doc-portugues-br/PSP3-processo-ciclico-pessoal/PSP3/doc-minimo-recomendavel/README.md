# Documentação Mínima Recomendável — PSP3 (Processo Cíclico Pessoal)

PSP3 escala o PSP para programas maiores dividindo o desenvolvimento em ciclos (iterações), cada um com seu planejamento, registros e revisões.

## Estrutura
- projeto/ — Requisitos e descrição do projeto
- iteracoes/ — Uma pasta por ciclo (iteracao-01, iteracao-02), cada uma com plano de trabalho, estimativa de esforço, registro de tempo e registro de defeitos
- qualidade/ — Checklists de design e código e classificação de defeitos
