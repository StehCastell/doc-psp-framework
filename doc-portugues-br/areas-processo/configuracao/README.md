# 🗂️ Área de Processo — Configuração

> Esta pasta é **autossuficiente**: tudo o que você precisa para entender a área está aqui.

## 📌 O que é esta área no PSP
Controle de versão dos seus artefatos pessoais (código, dados, documentos).

## 🎯 Qual problema ela resolve
Resolve **perder histórico e versões** do trabalho pessoal.

## ✅ O que entra nesta área
- Controle de versões pessoais
- Organização dos artefatos

## 🚫 O que NÃO entra nesta área
- Planejamento (→ planejamento)
- Relatórios (→ relatorios)
