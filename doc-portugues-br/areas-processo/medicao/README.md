# 📊 Área de Processo — Medição

> Esta pasta é **autossuficiente**: tudo o que você precisa para entender a área está aqui.

## 📌 O que é esta área no PSP
Coleta de dados pessoais: tempo, tamanho e defeitos. Base objetiva do PSP.

## 🎯 Qual problema ela resolve
Resolve o **não saber onde gasta tempo nem onde erra**.

## ✅ O que entra nesta área
- Registro de tempo
- Registro de defeitos
- Medição de tamanho
- Métricas de qualidade (yield, A/FR)

## 🚫 O que NÃO entra nesta área
- Estimativas (→ planejamento)
- Análise dos dados (→ analise)

## 📈 Como evolui do PSP0 ao PSP3

| Estágio PSP | Evolução da área |
|---|---|
| Estágio PSP0 — Linha de Base | Medição de tempo e defeitos. |
| Estágio PSP0.1 — Padrão e Tamanho | Acrescenta medição de tamanho. |
| Estágio PSP1 — Planejamento Pessoal | Dados de tamanho para estimativa. |
| Estágio PSP1.1 — Cronograma e Valor Agregado | Medição de valor agregado. |
| Estágio PSP2 — Qualidade Pessoal | Medição de defeitos por fase. |
| Estágio PSP2.1 — Projeto e Verificação | Métricas de qualidade (yield, A/FR). |
| Estágio PSP3 — Processo Cíclico | Medição consolidada entre iterações. |
