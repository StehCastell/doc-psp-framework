# 📅 Área de Processo — Planejamento

> Esta pasta é **autossuficiente**: tudo o que você precisa para entender a área está aqui.

## 📌 O que é esta área no PSP
Estimativas pessoais de tamanho e tempo (método PROBE) e cronograma.

## 🎯 Qual problema ela resolve
Resolve **estimativas no chute** que estouram prazos.

## ✅ O que entra nesta área
- Estimativa de tamanho (PROBE)
- Estimativa de esforço/tempo
- Cronograma
- Plano de trabalho

## 🚫 O que NÃO entra nesta área
- Dados brutos (→ medicao)
- Análise das estimativas (→ analise)
