# 📁 Área de Processo — Projeto

> Esta pasta é **autossuficiente**: tudo o que você precisa para entender a área está aqui.

## 📌 O que é esta área no PSP
Concentra os dados e o plano de cada projeto pessoal (resumo, descrição).

## 🎯 Qual problema ela resolve
Resolve **dados de projeto espalhados e sem consolidação**.

## ✅ O que entra nesta área
- Descrição do projeto
- Resumo do projeto
- Dados consolidados do projeto

## 🚫 O que NÃO entra nesta área
- Estimativas detalhadas (→ planejamento)
- Relatórios finais (→ relatorios)

## 📈 Como evolui do PSP0 ao PSP3

| Estágio PSP | Evolução da área |
|---|---|
| Estágio PSP0 — Linha de Base | Resumo do projeto e dados base. |
| Estágio PSP0.1 — Padrão e Tamanho | Projeto com padrão de codificação. |
| Estágio PSP1 — Planejamento Pessoal | Projeto com estimativas. |
| Estágio PSP1.1 — Cronograma e Valor Agregado | Projeto com cronograma. |
| Estágio PSP2 — Qualidade Pessoal | Projeto com plano de qualidade. |
| Estágio PSP2.1 — Projeto e Verificação | Projeto com design verificado. |
| Estágio PSP3 — Processo Cíclico | Projeto desenvolvido em ciclos. |
