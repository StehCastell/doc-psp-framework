# 📄 Área de Processo — Relatórios

> Esta pasta é **autossuficiente**: tudo o que você precisa para entender a área está aqui.

## 📌 O que é esta área no PSP
Comunica resultados: relatório de testes, resumo do ciclo, relatório final.

## 🎯 Qual problema ela resolve
Resolve **não fechar o ciclo de aprendizado** ao fim do trabalho.

## ✅ O que entra nesta área
- Relatório de testes
- Resumo de medições
- Relatório final do projeto

## 🚫 O que NÃO entra nesta área
- Coleta de dados (→ medicao)
- Análise (→ analise)

## 📈 Como evolui do PSP0 ao PSP3

| Estágio PSP | Evolução da área |
|---|---|
| Estágio PSP0 — Linha de Base | Relatório de testes e resumo do ciclo. |
| Estágio PSP0.1 — Padrão e Tamanho | Relatórios com dados de tamanho. |
| Estágio PSP1 — Planejamento Pessoal | Relatório de estimativas. |
| Estágio PSP1.1 — Cronograma e Valor Agregado | Relatório de planejamento e valor agregado. |
| Estágio PSP2 — Qualidade Pessoal | Relatório de revisões e defeitos. |
| Estágio PSP2.1 — Projeto e Verificação | Relatório de qualidade do design. |
| Estágio PSP3 — Processo Cíclico | Relatório consolidado das iterações. |
