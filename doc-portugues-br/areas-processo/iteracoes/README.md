# 🔁 Área de Processo — Iterações

> Esta pasta é **autossuficiente**: tudo o que você precisa para entender a área está aqui.

## 📌 O que é esta área no PSP
Desenvolvimento cíclico do PSP3: repetir o processo completo em iterações para escalar a projetos maiores.

## 🎯 Qual problema ela resolve
Resolve **aplicar o PSP em projetos grandes** que não cabem num único ciclo.

## ✅ O que entra nesta área
- Plano de trabalho por iteração
- Estimativas por iteração
- Revisões por iteração
- Relatório de iteração

## 🚫 O que NÃO entra nesta área
- Definição do processo (→ processo-pessoal)
- Consolidação final (→ relatorios)
