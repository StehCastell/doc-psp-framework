# 🧭 Área de Processo — Processo Pessoal

> Esta pasta é **autossuficiente**: tudo o que você precisa para entender a área está aqui.

## 📌 O que é esta área no PSP
Define e melhora o **seu** processo de trabalho: scripts, fases e propostas de melhoria (PIP). É o coração do PSP.

## 🎯 Qual problema ela resolve
Resolve trabalhar **sem método pessoal definido**, o que impede medir e melhorar.

## ✅ O que entra nesta área
- Definição do processo pessoal (scripts/fases)
- Padrão de codificação
- Propostas de melhoria de processo (PIP)
- Anotações do processo

## 🚫 O que NÃO entra nesta área
- Estimativas (→ planejamento)
- Dados de medição (→ medicao)

## 💡 Exemplos de uso
- Definir as fases do seu processo (planejar, projetar, codificar, revisar, testar).
- Registrar uma proposta de melhoria (PIP).

## 🔗 Relação com outras áreas
- **medicao** — o processo gera os dados.
- **qualidade** — revisões fazem parte do processo.
- **planejamento** — o processo é planejado.
