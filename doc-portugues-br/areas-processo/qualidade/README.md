# ✅ Área de Processo — Qualidade

> Esta pasta é **autossuficiente**: tudo o que você precisa para entender a área está aqui.

## 📌 O que é esta área no PSP
Gestão pessoal de defeitos: revisões de código e design, checklists, plano de qualidade.

## 🎯 Qual problema ela resolve
Resolve **defeitos que escapam para o teste e o cliente**.

## ✅ O que entra nesta área
- Revisão de código
- Revisão de design
- Checklists de revisão
- Plano de qualidade
- Gestão de defeitos

## 🚫 O que NÃO entra nesta área
- Medição bruta (→ medicao)
- Análise causal (→ analise)
