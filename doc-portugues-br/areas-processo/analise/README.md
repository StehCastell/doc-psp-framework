# 🔬 Área de Processo — Análise

> Esta pasta é **autossuficiente**: tudo o que você precisa para entender a área está aqui.

## 📌 O que é esta área no PSP
Converte medição em insight: precisão das estimativas, produtividade, tendências.

## 🎯 Qual problema ela resolve
Resolve **ter dados mas não aprender com eles**.

## ✅ O que entra nesta área
- Análise de estimativas
- Análise de produtividade
- Análise de defeitos
- Análise de tendências

## 🚫 O que NÃO entra nesta área
- Coleta (→ medicao)
- Ações de melhoria (→ processo-pessoal)
