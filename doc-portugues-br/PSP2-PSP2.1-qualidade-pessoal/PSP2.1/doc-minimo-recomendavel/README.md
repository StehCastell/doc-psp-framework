# Documentação Mínima Recomendável — PSP2.1 (Qualidade Pessoal)

PSP2.1 aprofunda a qualidade com classificação padronizada de defeitos (Defect Type Standard) e análise de defeitos.

## Estrutura
- projeto/ — Requisitos e descrição do projeto
- planejamento/ — Estimativa de tamanho, estimativa de esforço e plano de trabalho
- medicao/ — Registro de tempo, registro de defeitos e tamanho do software
- qualidade/ — Checklists de design e código, revisão de design, revisão de código, classificação de defeitos e análise de defeitos
