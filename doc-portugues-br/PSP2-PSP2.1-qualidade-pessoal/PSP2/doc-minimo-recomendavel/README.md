# Documentação Mínima Recomendável — PSP2 (Qualidade Pessoal)

PSP2 introduz a gestão de qualidade pessoal: revisões de design e de código com checklists.

## Estrutura
- projeto/ — Requisitos e descrição do projeto
- planejamento/ — Estimativa de tamanho, estimativa de esforço e plano de trabalho
- medicao/ — Registro de tempo, registro de defeitos e tamanho do software
- qualidade/ — Checklists de design e código, revisão de design e revisão de código
