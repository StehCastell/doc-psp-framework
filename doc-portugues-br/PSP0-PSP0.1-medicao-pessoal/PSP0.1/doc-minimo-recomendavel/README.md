# Documentação Mínima Recomendável — PSP0.1 (Medição Pessoal)

PSP0.1 adiciona padrão de codificação, medição de tamanho e proposta de melhoria de processo (PIP) à linha de base do PSP0.

## Estrutura
- projeto/ — Requisitos e descrição do projeto
- medicao/ — Registro de tempo, registro de defeitos e tamanho do software
- processo-pessoal/ — Padrão de codificação e melhorias de processo
