# Documentação Completa — PSP1 (Planejamento Pessoal)

Conjunto completo do PSP1.

## Estrutura
- projeto/ — Requisitos, descrição do projeto e anotações
- planejamento/ — Estimativa de tamanho, estimativa de esforço, plano de trabalho, cronograma e acompanhamento
- medicao/ — Registro de tempo, registro de defeitos, tamanho do software e resumo das medições
- processo-pessoal/ — Padrão de codificação, fluxo de desenvolvimento, melhorias de processo e lições aprendidas
- configuracao/ — Controle de mudanças
- relatorios/ — Relatório planejado vs realizado e relatório de encerramento
