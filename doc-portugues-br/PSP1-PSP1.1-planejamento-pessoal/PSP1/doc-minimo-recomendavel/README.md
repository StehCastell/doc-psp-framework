# Documentação Mínima Recomendável — PSP1 (Planejamento Pessoal)

PSP1 adiciona planejamento ao processo: estimativa de tamanho e esforço (método PROBE) e plano de trabalho.

## Estrutura
- projeto/ — Requisitos e descrição do projeto
- planejamento/ — Estimativa de tamanho, estimativa de esforço e plano de trabalho
- medicao/ — Registro de tempo, registro de defeitos e tamanho do software
- processo-pessoal/ — Padrão de codificação e melhorias de processo
