# Documentação Completa — PSP1.1 (Planejamento Pessoal)

Conjunto completo do PSP1.1.

## Estrutura
- projeto/ — Requisitos, descrição do projeto e anotações
- planejamento/ — Estimativa de tamanho, estimativa de esforço, plano de trabalho e cronograma
- medicao/ — Registro de tempo, registro de defeitos, tamanho do software e resumo das medições
- analise/ — Histórico de produtividade, histórico de estimativas, planejado vs realizado, análise de estimativas e relatório de produtividade
- processo-pessoal/ — Padrão de codificação, melhorias de processo e lições aprendidas
- relatorios/ — Relatório de encerramento (postmortem)
