# Documentação Mínima Recomendável — PSP1.1 (Planejamento Pessoal)

PSP1.1 acrescenta análise de dados históricos (produtividade) e comparação planejado vs realizado ao planejamento do PSP1.

## Estrutura
- projeto/ — Requisitos e descrição do projeto
- planejamento/ — Estimativa de tamanho, estimativa de esforço e plano de trabalho
- medicao/ — Registro de tempo, registro de defeitos e tamanho do software
- analise/ — Histórico de produtividade e planejado vs realizado
